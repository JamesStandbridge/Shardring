# Systeme Progression

## Role

La progression gere ce qui survit entre les runs. Elle doit rester separee de l'economie de run.

## Contrat futur

- Stocker une monnaie cross-game rare.
- Debloquer plus tard du contenu ou des options persistantes.
- Utiliser un format de sauvegarde versionne et testable.
- Ne jamais bloquer la boucle de run de base si une sauvegarde est absente ou invalide.

## Limites actuelles

Aucune sauvegarde persistante n'est implementee maintenant. Les besoins sont documentes pour eviter de melanger trop tot run et meta progression.
